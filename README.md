# 项目运行
> 运行前请先去官网安装nodejs

> 打开命令行工具

> clone项目到本地(在命令行工具输入以下代码)
 ```
 git clone git@github.com:yeyan1996/movieApp.git
```
> 切换到movieApp文件夹
 ```
 cd movieApp
```
> 安装项目的所有依赖
 ```
 npm install
```
> 运行webpack.dev.server
```
npm run dev
```
